# 📡 WatchTower

Software para observação e executação de “exames” periódicos em servidores — monitorando processos, serviços e métricas importantes para automação de manutenção e alertas.watchtower

## 🧠 Sobre

**Esse projeto foi criado com o objetivo de fornecer uma ferramenta leve em Python para verificar automaticamente o estado de serviços e servidores, permitindo:**

 - Verificar condições periódicas de sistemas.

 - Registrar logs e gerar alertas (Exportando relatórios).

 - Automatizar inspeções de saúde dos seus servidores.

***📌 Esse repositório ainda não possui uma documentação detalhada de uso — sinta-se à vontade para abrir issues, melhorias ou contribuições!***

## 🚀 Funcionalidades

***O WatchTower pode ser usado para:***

 - 🔎 Executar verificações em servidores ou serviços.

 - 📊 Registrar status de verificações em logs.

 - 🛠️ Automatizar verificações de serviços em Python.

 - 🐍 Baseado em Python, com dependências leves.

## 🧰 Tecnologias

**O projeto foi feito com:**

 - Python 3.x

 - Bibliotecas listadas em ***requirements.txt***]
   
# 🔧 Instalação

**1. 📝 Clone o repositório:**
```
git clone https://github.com/Arajojo/watchtower.git
```

**2. 📁 Entre na pasta:**
```
cd watchtower
```

**3. 💽 Crie e ative um virtualenv (opcional, mas recomendado):**
```
python -m venv venv
source venv/bin/activate  # Linux / macOS
venv\Scripts\activate     # Windows
```

**4. 📲 Instale as dependências:**
```
pip install -r requirements.txt
```

## ▶️ Uso

Execute o script principal:
```
python main.py
```
***O programa por enquanto não possui nenhum argumento.***
